# Full Cycle - Docker challenges

## Go challenges

--------------------------------------------
#### Run from dockerhub
--------------------------------------------

```
docker run --rm -it eriksonbezerra/go_hello
```

--------------------------------------------
#### Create your own image
--------------------------------------------

<br />

Build the image, run this command:

```
docker build -t go_app_hello .
```

Run your container image

```
docker run go_app_hello
```
